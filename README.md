Comparador de Preços

Um projeto simples desenvolvido com **HTML** e **JavaScript** que permite comparar o preço de dois produtos e informar qual é a melhor opção de compra, além da economia obtida.

## Demonstração

O usuário informa o preço de dois produtos e o sistema:

-  Verifica se os valores foram preenchidos;
-  Informa qual produto é mais barato;
-  Calcula quanto será economizado;
-  Informa quando os dois produtos possuem o mesmo preço.

## Tecnologias Utilizadas

- HTML5
- JavaScript (ES6)

- Como Executar

1. Faça o download ou clone o repositório:

```bash
git clone https://github.com/Plaads/Comparadorsite.git
```

2. Abra o arquivo **comparador.html** em qualquer navegador.

Não é necessário instalar nenhuma biblioteca ou framework.

## 💻 Exemplo

### Entrada

```
Produto 1: R$ 18,90
Produto 2: R$ 25,40
```

### Saída

```
Produto 1 é mais barato.
Você economiza R$ 6,50.
```
